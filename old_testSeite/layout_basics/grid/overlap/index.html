Grid Overlap

Damit sich die beiden Grid-Elemente überlagren können, müssen im Css gleichzeitig aufgerufen werden.

Die Zeile .gImage, .gContent{grid-row: 1 / 3;} weist beiden Elementen gleichzeitige die gleiche Reihe zu

Außerdem war es nötig dem Paragraph die Anweisung: align-self: mitzugeben, damit sich sein Padding nicht sinnlos ausbreitet. 

.gridContainer{
  width: 600px;
  margin: 20px auto;
  border: 3px solid snow;
  display: grid;
  grid-template-columns: repeat(5, 1fr);
} 
.gImage{
  grid-column: 1 / 3;
}
.gImage, .gContent{
  grid-row: 1 / 3;
}

.gContent{
  grid-column: 2 / -1;
  align-self: flex-start;
  background-color: snow;
  padding: .25em;
  box-shadow: 4px 4px 10px rgba(0, 0, 0, .25);
}
