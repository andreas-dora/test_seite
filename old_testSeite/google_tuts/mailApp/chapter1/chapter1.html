Einleitung

Mit MailApp kann man aus einer Google Tabelle Email versenden, auch wenn das Google-Konto über keine Email-Adresse verfügt.

MailApp kann nicht auf den Posteingang zugreifen. Als Absender wird die Emailadresse angegeben, auf die dein Google-Konto läuft. So kannst Du Post empfangen und bekommst auch Fehlermeldungen, falls eine Nachricht nicht übermittelt werden kann.

Hat dein Google-Konto eine Gmail-Adresse solltest du anstelle von MailApp GmailApp benutzen.

Möglichkeiten:

MailApp kann benutz werden um Inhalte einer Tabelle direkt zu verschicken oder kleine RundMails zu verfassen.

Limitierung:

Bei einem kostenlosen Google-Account ist die Anzahl der Nachrichten die man pro Tag verschicken auf 100 limitiert.

  • Betreff

Mit + lassen sich die Strings für den Body verbinden. Benötigt man ein Leerzeichen zwischen zwei Strings gibt man +“ ” ein. Einen Zeilenumbruch kann man mit \n erzeugen. Das hab ich mir mal die Variable nZ gepackt.

const nZ ="\n"

So kann ich meinen Text halbwegs sauber verschicken.

  MailApp.sendEmail(kontakt, betreff, anrede + nZ + text + nZ + gruss + nZ, meinName);

Möchte man seine Nachricht weiter gestallten, muss man auf den erweiterten Funktionsumfang von sendMail zurückgreifen und einen htmlBody einfügen.

Hier noch mal der komplette Code:

function simpleMail() {
  
  const ss = SpreadsheetApp.getActiveSpreadsheet(); // Auf das verwendete Google Tabellen Datei zugreifen
  const tab = ss.getSheetByName("One"); // Wählt die gewünschte Tabelle (tab) aus.
  
  const nZ ="\n"; // Erzeugt in einem Javaskript-String eine neue Zeile.
  
  const kontakt = tab.getRange("B1").getValue(); // Mailadresse des Empfängers

MailApp.sendEmail(kontakt, betreff, anrede + nZ + text + nZ + gruss + nZ, meinName);
