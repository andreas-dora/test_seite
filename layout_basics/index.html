Some basic Layouts

Layouts mit mehreren Spalten

Css3 hat ein paar schöne Properties um dynamische Layouts mit mehreren Spalten zu erzeugen. Laut w3 School ist der Browser-Suport sehr gut. Lediglich column-span wird von. nicht unterstützt.

  • column-count: Anzahl der Spalten
  • column-gap: Abstand zwischen den Spalten
  • column-rule-style
  • column-rule-color
  • column-rule-width
  • column-rule: Funktioniert wie boarder und vereint die Werte von column-rule-style, -width und color
  • column-fill: balanced (oder auto): gibt an ob die Spalten gleichmäßig gefüllt werden.
  • column-width
  • column-span: Legt ein Element über mehrere Spalten.

Mit Außnahem von column-span werden alle Props auf den Mutter Container angewendet. Mit column-span

Flexbox

Flexbox

Grundlagen des Flexible Box Layouts. Um Störungen zu vermeiden, setzen wir im Body margin: 0;. Die Angabe display:flex; im Mutter-Container löst den Zauber aus. Beispiel 1 zeigt ein horizontales Flex Box Layout flex-flow:row wrap;. Die Angabe wrap sorgt dafür, dass die Elemente auch untereinander fließen, wenn in der Vertikalen kein Platz mehr ist.

Flex Direction

Wie eben erwähnt braucht es um Flexbox anwenden zu können immer einen Mutter-Container der display: flex; ausruft. Was dann auf alle Child-Elemente angewendet wird

Auf der Htlm-Seite haben wir im Moment nicht mehr als: 



<body>
<div class="parent">
    <div class="child" id="child1">1</div>
    <div class="child" id="child2">2</div>
    <div class="child" id="child3">3</div>
</div>
</body>

Auf der CSS_seite gibt es folgendes zu sehen:


    *{
        margin: 0px;
        padding: 0 px;
        box-sizing: border-box;
    }   
    body{
        margin: 0px;
        font-family: 'Roboto', sans-serif; 
        background: dimgray;
        font-size: 90px;
        color: dimgray;
    } 
    .parent{
        height: 100vh;
        border: 20px solid #000000; 
        display: flex;
        justify-content: space-evenly;
    }
    .child{
        width: 200px;
        height: 200px;
    }
    #child1{
        background-color: darkviolet;
    }
    #child2{
        background-color: violet;
    }
    #child3{
        background-color: mediumvioletred;
    }

Im Mutter-Container hier .class wird mit display:flex die Sachen ins Rollen gebracht. Für Strech darf dem Kind keine Größe zugewiesen sein. Dann kann sich das Element in der Vertikalen ausdehnen. In diesem Beispiel werden die Flexbox Properties gründlich analysiert. 

                        flex-direction: row-reverse;

Felx-Direction kann row, column und row-reverse bzw column-reverse


Justify-content

Zum anfang oder zum Ende mit Justify-content: start und end. 

justify-content: space-around;


.parent{
    width: 400px;
    height: 400px;
    border: 2px solid mediumvioletred;
    margin: 20px auto; 
    display: flex;
    align-items: baseline;
}

flex: ist die Shorthand Version für felx-grow, flex shrink und flex- basis. mit bais wird width nicht mehr benötigt
