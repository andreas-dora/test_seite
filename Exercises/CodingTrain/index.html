First Server with Node

Dieses Video hilft: Coding Train 8.2: HTTP Server with Express.

Erste Schritte:
  • Folder mit index.js erstellen
  • im Ordenr den Befehl: npm initaufrufen
  • Als Entry-Point die Datei index.js bestimmen

    • in index.js kann man sich eine minimale App bauen wie zB auf der Express-Homepage beschrieben.


    				
						var express = require('express');
						var app = express();
						
						app.get('/', function (req, res) {
							res.send('Hello World!');
						});
						
						app.listen(3000, function () {
							console.log('Example app listening on port 3000!');
						});

    oder


    				
						const express = require('express');
						const app = express();
						app.listen(3000, () => console.log('listening at 3000'));
						app.use(express.static('public'));

    Diese kann man anscchließend mit node app.js starten. Die heßt nicht wie die Datei, sondern wie die Funktion in der Datei! \n Um zu einem sichtbaren Ergebnis zu kommmen darf man nicht vergessen noch eine html-Seite einzufügen. Dafür erstellt man sich am besten eine "public" folder.
