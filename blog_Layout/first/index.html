Googles AppScript
Dies ist ein Font Test. Er dient lediglich dazu mir etwas mehr Überblick zu verschaffen. Überblick könnnen wir alle dringend gebrauchen, oder glaubst du das etwa nicht?

Den Benutzern von Googles Office Produkten, der GSuite, steht mit AppScript ein Werkzeug zur Verfügung das es ermöglicht Arbeitsprozesse zu automatisieren. AppScript Ist das Pendant zu Microsofts Visual Basic. Die verwendete Sprache ist hier zeitgemäß Javascript. Für alle GSuite Anwendungen und noch viel mehr stehen in AppScript Dienste zur bereit, mit denen man auf die Funktionalität der Anwendung zugreifen kann.

Der Skirpteditor

In den einzelnen Anwendungen ist im Menü Tools der Punkt Skripteditor zu finden. Für den Anfang möchte ich mich auf Tabellen konzentrieren. Also bitte mal eine frische Tabelle öffnen, dem Ding einen Namen geben und den Skripteditor öffnen. Schwups, findet man sich in einem Text-Editor wieder. Wie bereits erwähnt hat man es hier mit Javaskript zu tun. In der Welt von Google gibt es natürlich ein paar Besonderheiten. Anstelle von cosole.log() benutzt man hier Logger.log(). 

function myFunction() {
    Logger.log("Was geht ab?");
  }

Hat man den Code gespeichert, kann er über die “Play”-Taste ausgeführt werden. Die Ausgabe des Loggers findet man im Menü Ansicht unter Protokolle.

Ein- und auskommentieren

Das ein- und auskommentieren funktioniert, wie in Javascript gewohnt mit Doppelslash für einzeilige Kommentare und Slash plus Sternchen für mehrzeilige. 


/*
Das ist mein Code und
das ist ein mehrzeiliger
Komentar!
*/
function myFunction() {
  const integer_1 = 17;
  const integer_2 = 4;
  const result = integer_1 + integer_2; // hier wird gerechnet
  Logger.log(result); 
// Dieser Kommentar ist unnötig
}

Syntax Highlighting

Der Online Editor für AppSkript ist nicht sonderlich komfortabel. Jede klammer die man öffnet, muss man auch selbst wieder schließen. Aber immerhin ist das Syntax Highlighting recht hilfreich. Variablen werden beim Initialisieren dunkelblau dargestellt, und später hellblau. Strings sind rot, Zahlen egal ob Integer oder Float sind grün. Auskommentiertes wird braun wiedergegeben. In Zeile 13 hab ich mich ein wenig vertippt. Die falsch geschriebene Variable wird nicht blau dargestellt.

Die Benutzeroberfläche ist weitestgehend selbsterklärend. Unter dem Hauptmenü findet man Vignetten für: - Rückgängig - Ne, doch nicht - Einrücken - Speichern - Trigger hinzufügen - Ausführen - Funktionsauswahl
