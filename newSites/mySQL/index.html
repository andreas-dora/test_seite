Eine erste Abfrage

  • Mit CMD + T eine neue Skriptdatei ereugen.
  • Folgende Eingabe: SELECT name,vorname FROM mitarbeiter;
  • Ausführen mit dem Blitzsymbol oder CMD + ENTER

Kommentare hinzufügen

Einzeilige Kommerntare werden mit doppeltem Bindestrich eingeleitet:

-- Mein Kommentar

Mehrzeilige Kommentare werden wie in JS zwischen Slah und Sternchen gesetzt:

/* Mein mehrzeiliges KOmmentar */

SQL Skripts sollten immer folgende Kommentare enthalten:

  • Angeforderte Stelle
  • Funktion der Anfrage
  • Datum der Fertigstellung
  • Durchführung von Änderungen und GRund der Änderung
  • Autor

/*
Autor: Andreas Breitwieser
Fertigstellung: 01.02.2021
Funktion: Liste der Namen und Vornamen der Mitarbeiter
Funktion geprüft: ja
Funktion erfüllt: ja
Nutzergruppen der Abfrage: Personalabteilung
*/
 -- Spaltenauswah: name, vornamee der Tabelle mitarbeiter
 SELECT name,vorname, abteilung FROM mitarbeiter WHERE abteilung = 'Einkauf';
