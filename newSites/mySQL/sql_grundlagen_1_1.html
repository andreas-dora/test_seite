SELECT & FROM

Hinter dem Schlüsselwort SELECT folgen die Spaltenbezeichnungen. Hinter dem Schlüsselwort FROM wird die Tabelle genannt auf die die Datenbank zugreifen soll. Wird mehr als eine Spalte abgefragt, werden diese mit Komma getrennt:

SELECT name, vorname FROM mitarbeiter;

Mit Sternchen werden alle Spalten einer Tabelle ausgewählt:

SELECT * FROM mitarbeiter;

Eine Auswahl von Spalten einer Tabelle wird Projektion genannt. (Ausgewählte Spalten werden in Tabellenform projiziert.)

Filtern mit WHERE

  • Numerische Werte vergleichen
  • Texte vergleichen
  • Textmuster vergleichen
  • Datumsangaben vergleichen
  • auf Intervalle prüfen
  • auf Existenz in Teilmengen prüfen
  • auf “Wert wurde definiert” prüfen

Vergleichsoperatoren

Operator Funktion
a=b ist gleich
a<>b ist nicht gleich
a!=b ist nicht gleich
a<b kleiner als
a<=b kleiner oder gleich
c BETWEEN a AND b  ob Wert c zwischen ab und b liegt
c NOT BETWEEN a AND b  ob c nicht zwischen ab und b liegt
a LIKE Textmuster ob a dem Textmuster entspricht
a NOT LIKE Textmuster ob a dem Textmuster nicht entspricht

Spalten auf Gleichheit prüfen

Aus der Mitarbeiter Tabelle sollen die Telefonnummern der von ‘Frau Lang’ herausgesucht werden:

SELECT
     name,
     vorname,
     festnetz,
     mobilnetz
 FROM
     mitarbeiter
 WHERE
     name = 'Lang';

Es sollen alle Mitarbeiter herausgesucht werden die in einem bestimmten Gebiet wohnen (PLZ):

SELECT
     name,
     vorname,
     plz
 FROM
     mitarbeiter
 WHERE
 plz = 50000;

Ungleicheit

Sucht nach Mitarbeitern deren Steuerklasse nicht 3 ist.

sql SELECT name, vorname, steuerklasse, verheiratet FROM mitarbeiter WHERE steuerklasse<>3;

Nach späterem Eintrittsdatum filtern

Daten werden zwar wie Strings mit einfachen Anführungszeichen versehen, lassen sich aber dennoch mit Größer- und Kleiner-Operatoren bearbeiten. Hier wird nach Mitarbeiter gesucht die am 01.01.90 oder danach eingestellt wurden.

SELECT
     name,
     vorname,
     bonus,
     eintrittsdatum
 FROM
     mitarbeiter
 WHERE
     eintrittsdatum>='1990-01-01';
