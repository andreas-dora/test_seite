Mit SQL Rechnen

Es gilt Punkt vor Strichrechnung. Es stehen die üblichen Operatoren zur Verfügung sowie der Modulus-Operator: %.

Ausdrücke/ Expressions

In SQL werden sogenannte Ausdrücke verwendet. Hierbei kann es sich um Rechenoperationen, oder um Bedingungen handeln, die ausgewertet werden. Ausdrücken ist gemein, dass sie ausgewertet werden und zu einem Ergebnis führen.

Ausdrücke können überall da verwendet werden, wo Werte erwartet werden. Daher können Sie Ausdrücke auch in zahlreichen anderen SQL-Anweiseungen (zB INSERT oder UPDATE) einsetzen.

SQL kennt sechs verschiedene Expressions:

  • Werte
  • Variablen
  • Berechnungen
  • Konstanten
  • Operationen
  • Vergleiche

Spaltenwerte

SELECT
    name, 
    vorname,
    bonus,
    monatslohn,
    bonus + monatslohn AS summe
FROM mitarbeiter;

Ergebnis:

name vorname bonus monatslohn summe
Müller Ralf 500.00 4000.00 4500.00
Schneider Petra 500.00 4000.00 4500.00
Klein Thomas 500.00 4000.00 4500.00
Lang Ute 280.50 2800.00 3080.50

Es können auch mehr als zwei Summanden verwendet werden:

SELECT
    name, 
    vorname,
    bonus,
    monatslohn,
    1000 AS wert1000,
    bonus + monatslohn + 1000 AS summe
FROM mitarbeiter;

Ergebnis:

name vorname bonus monatslohn wert1000 summe
Müller Ralf 500.00 4000.00 1000 4500.00
Schneider Petra 500.00 4000.00 1000 4500.00
Klein Thomas 500.00 4000.00 1000 4500.00
Lang Ute 280.50 2800.00 1000 3080.50

Multiplizieren

SELECT
    name,
    vorname,
    stundenpromonat,
    stundenlohn,
    stundenpromonat*stundenlohn AS gehalt
FROM
    mitarbeiter;

Ergebnis:

name vorname stundenpromonat stundenlohn summe
Müller Ralf 160 20.80 3328.00
Schneider Petra 180 28.50 5130.00
Klein Thomas 165 20.80 3432.00
Lang Ute 170 20.00 3400.00

Nach Ergebnissen filtern

SELECT
    name,
    vorname,
    stundenpromonat*stundenlohn AS monatslohnberechnet,
    monatslohn
FROM
    mitarbeiter
WHERE
    stundenpromonat*stundenlohn<monatslohn;

Ergebnis:

name vorname monatslohnberechnet monatslohn
Müller Ralf 3328.00 4000.00
Klein Thomas 3432.00 4000.00

Nach Ergebnis sortieren:

SELECT
    name,
    vorname,
    bonus,
    monatslohn,
    bonus - monatslohn AS sonderbonus
FROM
    mitarbeiter
ORDER BY
    bonus - monatslohn
DESC;
name vorname bonus monatslohn sonderbonus
Müller Ralf 200000.00 1800.00 4500.00
Schneider Petra 50000.00 6000.00 44000.00
Klein Thomas 10000.00 6000.00 4000.00
Lang Ute 10000.00 7500.00 2500.00
