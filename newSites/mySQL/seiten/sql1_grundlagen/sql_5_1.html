Hier sorgt die Anweisung UNION für ein eindeutiges Ergebnis. Jedes Element wird nur einmal übernommen. “Nur eindeutige Elemente werden zugelassen”.

SELECT element FROM mengea
        UNION
        SELECT element FROM mengeb;
