Key Events

Beim Benutzen einer Taste entstehen drei Events:

  • keydown
  • keypress
  • keyup

keydown erkennt Shift, alt, control und comand, keypress kann das nicht. Dementsprechend ist es nur keydown möglich Versalien, und den größten Teil der Sonderzeichen zu erkennen.

Codes

Bei jedem Tastaturanschlag KeyCode, key, code und location abgefragt werden. Informationen auf Grund von keyCodes entgegen zu nehmen gilt als veraltet, weil das System an einigen Stellen zu ungenau war. Key gibt die Werte der lokalen Tastatur wieder. Da es recht viele unteschiedliche Länder-Tastaturen gibt - Schweizer haben kein 'ß' - ist diese Methode für Webanwedungen auch nicht empfelenswwert.

code bezieht sich immer auf die amerikanische Tastatur (qwerty). Lässt man sich von der Tasturbelegung nicht verwirren, wird man mit sehr eindeutigen Ausgaben wie KeyM und Digit9 belohnt. Diese Eindeutigkeit macht das System zur besten alternative für die Eventsteuerung.
