Elemente hinzufügen: push(), unshift()

Zum stehen mehrere Methoden zur Verfügung um Elemente einem Array hinzuzufügen.

Elemente am Ende hinzufügen mit push()

 
  var todoList = [];
  var plusPunkt;
  
  plusPunkt = todoList.push('Bad putzen');
  console.log(plusPunkt);
  
  plusPunkt = todoList.push('Einkaufen');
  console.log(plusPunkt);
  
  plusPunkt = todoList.push('Rasen mähen');
  console.log(plusPunkt);
  
  plusPunkt = todoList.push('Kochen');
  console.log(plusPunkt);
  
  console.log(todoList);

Elemente am Anfang hinzufügen mit unshift()

 var todoList = [];
      var plusPunkt;

Einfügen, ersetzen und Löschen splice()

Die Methode splice() ist äußerst vielseitig. Sie kann genutzt werden um Elemente in ein Array Einzufügen, Elemente zu löschen oer Elemente zu Ersetzen. Die Methode erwartet 2 bis 3 Argumente:

  1. Die Stelle an der Eingefügt, gelöscht oder Ersetzt werden werden soll.
  2. Die Anzahl der Elemente, die gelöscht werden sollen. Mit 0 wird kein Element gelöscht.
  3. Optional: das Element, dass eingefügt werden soll

var todoList = [
      'Frühstück',
      'Einkaufen',
      'Aufräumen',
      'Coding'
    ];
    
    todoList.splice(3, 0, 'Daten sichern');
    console.log(todoList);

Einfügen, ersetzen und Löschen splice()

splice() kann genutzt werden um an einer bestimmten Stelle des Arrays ein oder mehrere Elemente zu löschen. Die Methode erwartet 2 bis 3 Argumente:

  1. Die Stelle an der Eingefügt, gelöscht oder Ersetzt werden werden soll.
  2. Die Anzahl der Elemente, die gelöscht werden sollen. Mit 0 wird kein Element gelöscht.
  3. Optional: das Element, dass eingefügt werden soll

Einen bestimmten Teil mit splice() entfernen

var todoList = [
      'Frühstück',
      'Einkaufen',
      'Aufräumen',
      'Coding'
    ];
    
    todoList.splice(3, 1);
    console.log(todoList);

Mit splice() eins hinzufügen und zwei löschen

var todoList = [
      'Frühstück',
      'Einkaufen',
      'Aufräumen',
      'Coding',
      'Workout'
    ];
    
    var item1 = todoList.splice(2, 2, 'Faulenzen');
    console.log(todoList);

Teile kopieren: slice()

Mit slice() kann man einen ausgewählten Teil kopieren;
slice() erwartet ein bis zwei Argumente. 1. Anfang der Auswahl 2. Das Ende der Auswahl (Die Auswahl endet davor)

var todoList = [
      '0. Aufstehen',
      '1. Frühstück',
      '2. Einkaufen',
      '3. Aufräumen',
      '4. Daten sichern',
      '5. Coding',
      '6. Datenabgleich',
      '7. Workout'
    ];
    
    var teil = todoList.slice(2);
    console.log(teil);

Das Ende der Auswahl schließt vor dem im zweiten Argument übergebenen Wert.

var teil = todoList.slice(2,5);
      console.log(teil);

Gibt aus dem obigen Array die Punkte 2., 3. und 4. zurück

Es können auch negative Werte übergeben werden

Übernimmt den dritten bis vorletzten Eintrag.

var teil = todoList.slice(2,-1);

Sortieren: sort(), reverse()

sort() ist eine Higher Order Funktion. Solche Funktionen arbeiten mit Vergleichsfunktionen. Ohne Vergleichsfunktion wird das Array nach Wertigkeit der ASCII-Zeichen durchsortiert.

  1. 404
  2. 5
  3. 'Aufräumen'
  4. 'Workout'
  5. 'arbeiten'
  6. 'Übung'
var todoList = [
          'arbeiten',
          5,
          404,
          'Aufräumen',
          'Workout',
          'Übung'
          ];
        
        todoList.sort();
        console.log(todoList);

gibt reverse() den Inhalt eines Array einfach in umgekehrter Reihenfolge wieder.

sort() ermöglicht es ein Array anhand einer übergebenen Vergleichsfunktion zu sortieren.

var werte = [
    7, 9, 6, 3, 8, 2, 4, 1, 5
  ];
  
  function compare(wert1, wert2){
    if (wert1 < wert2){
      return -1;
    }
    else if(wert1 > wert2){
      return 1;
    }
    else {
      return 0;
    }
  }
  werte.sort(compare);
  console.log(werte);

Beispiel für alphabetische Sortierung

var contacts = [
    {
      firstName: 'Britta',
      lastName: 'Master',
      email: 'britta@test.de'
    },
    {
      firstName: 'Andi',
      lastName: 'Dora',
      email : 'andi.dora@test.de'
    },
    {
      firstName: 'Elira',
      lastName: 'Klaviara',
      email: 'elvira@klaviara.de'
    }
  ]
  
  function compareByFirstName(contact1, contact2){
    return contact1.firstName.localeCompare(contact2.firstName)
  }
  
  contacts.sort(compareByFirstName);
  console.group(contacts);
