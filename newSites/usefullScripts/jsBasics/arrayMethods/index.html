Alphabetische Übersicht

Methode Beschreibung
concat() Hängt Elemente (oder Arrays) an
filter() Filtert Elemente heraus
forEach() Wendet eine Funktion auf alle Elemente an
join() Wandelt ein Array in einen String um
map() Bildet die Elemente auf Basis einer Umwandlungfunktion ab
pop() Entfernt das letzte Element
push() Fügt am Ende ein neues Element hinzu
reduce() Auf Basis einer übergebenen Funktion werden alle Werte zusammen gefasst
revers() Kehrt die Reihenfolge der Elemente in einem Array um
shift() Entfernt das letzte Element eines Arrays
slice() Schneidet ein einzelnes Element heraus
splice() Fügt ein neues Element an einer bestimmten Position ein
sort() Sortiert ein Array auf Basis einer Funktion

Elemente hinzufügen

Elemente am Ende hinzufügen mit push()

 
 var todoList = [];
var plusPunkt;

plusPunkt = todoList.push('Bad putzen');
console.log(plusPunkt);

plusPunkt = todoList.push('Einkaufen');
console.log(plusPunkt);

plusPunkt = todoList.push('Rasen mähen');
console.log(plusPunkt);

plusPunkt = todoList.push('Kochen');
console.log(plusPunkt);

console.log(todoList);

Elemente am Anfang hinzufügen mit unshift()

 var todoList = [];
var plusPunkt;

plusPunkt = todoList.push('Bad putzen');
plusPunkt = todoList.push('Einkaufen');
plusPunkt = todoList.push('Rasen mähen');
plusPunkt = todoList.push('Kochen');

plusPunkt = todoList.unshift('Frühstück');

console.log(todoList);

Mit splice() Elemente an einer bestimmten Position hinzufügen

Die Methode splice() erwartet 2 bis 3 Argumente: 1. Die Stelle an der Eingefügt werden soll. 2. Die Anzahl der Positionen die gelöscht werden sollen 3. Den neuen Inhalt

var todoList = [
  'Frühstück',
  'Einkaufen',
  'Aufräumen',
  'Coding'
];

todoList.splice(3, 0, 'Daten sichern');
console.log(todoList);

Teile kopieren

Mit slice() kann man einen ausgewählten Teil kopieren;
slice erwartet ein bis zwei Argumente. 1. Anfang der Auswahl 2. Das Ende der Auswahl (Die Auswahl endet davor)

var todoList = [
  '0. Aufstehen',
  '1. Frühstück',
  '2. Einkaufen',
  '3. Aufräumen',
  '4. Daten sichern',
  '5. Coding',
  '6. Datenabgleich',
  '7. Workout'
];

var teil = todoList.slice(2);
console.log(teil);

Das Ende der Auswahl schließt vor dem im zweiten Argument übergebenen Wert.

var teil = todoList.slice(2,5);
console.log(teil);

Gibt aus dem obigen Array die Punkte 2., 3. und 4. zurück

Es können auch negative Werte übergeben werden 

var teil = todoList.slice(2,-1);

Übernimmt den dritten bis vorletzten Eintrag.

Sortieren

Für das Sortieren der Elemente eines Arrays stehen zwei Methoden zur Verfügung.

reverse() gibt den Inhalt eines Array einfach in umgekehrter Reihenfolge wieder.

sort() ermöglicht es ein Array anhand einer übergebenen Vergleichsfunktion zu sortieren.

var werte = [
  7, 9, 6, 3, 8, 2, 4, 1, 5
];

function compare(wert1, wert2){
  if (wert1 < wert2){
    return -1;
  }
  else if(wert1 > wert2){
    return 1;
  }
  else {
    return 0;
  }
}

werte.sort(compare);
console.log(werte);

Beispiel für alphabetische Sortierung

var contacts = [
  {
    firstName: 'Britta',
    lastName: 'Master',
    email: 'britta@test.de'
  },
  {
    firstName: 'Andi',
    lastName: 'Dora',
    email : 'andi.dora@test.de'
  },
  {
    firstName: 'Elira',
    lastName: 'Klaviara',
    email: 'elvira@klaviara.de'
  }
]

function compareByFirstName(contact1, contact2){
  return contact1.firstName.localeCompare(contact2.firstName)
}

contacts.sort(compareByFirstName);
console.group(contacts);
