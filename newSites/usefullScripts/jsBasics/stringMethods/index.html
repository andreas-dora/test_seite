Einen String durchsuchen

Wie Arrays können String mit indexOf() und lastIndexOf() durchsucht werden. Im Gegensatz zu Arrays kann man Strings auch auf einzelne Zeichen oder Teilstrings durchsuchen.

Der Methode indexOf() kann optional ein zweites Argument übergeben werden um nach einem bestimmten (hier zweiten) Auftreten von ‘M’ zu suchen:

 var userName = "Max Mustermann";
console.log(userName.indexOf('M', 2));

Case sensitive

Die Suche in einem String ist ‘case sensitive’. Daher lohnt es sich einen ‘Arbeits-String’ zu erzeugen in dem alle Zeichen klein geschrieben sind bevor man auf die Suche geht.

var userName = "Max Mustermann";
var klein = userName.toLowerCase();
console.log(klein.lastIndexOf('m'));

Mit toUpperCase() wandelt man einen String in Großbuchstaben um.

Einen Teil-String erzeugen

Zum erzeugen von Substrings stehen drei Methoden zur verfügung: - slice() - substring() - substr()

Den Methoden kann neben dem Start-Index optional ein zweiter Wert übergeben werden. Bei slice() und substring() ist das der End-Index, bei substr() gibt man die gewünschte Länge (Anzahl der Zeichen) an

var userName = "Max Matthias Mustermann";
var lastName = userName.substring(userName.lastIndexOf(' ')+1, userName.lenght);
console.log(lastName);
