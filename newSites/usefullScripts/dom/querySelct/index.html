Query Selector Methoden


by ID

document.querySelector('#test').innerText ="Hey!";

by Class

document.querySelector('.btn').className;

Die Methode spukt einen Unübersichtlichen String ausKlassen aus auf 'btn' zutrift'

der schaut so aus: "btn inlineBtn" Auf die Frage nach der .classList bekommt man die Array, die DOM Token List 

DOMTokenList(2)
            0: "btn"
            1: "inlineBtn"
            length: 2       ​
            value: "btn inlineBtn"

Das Ergeniss ist nicht nur Übersichtlicher, man kann die Elemente des Arrays lassen sich aus einem Skript auch leichter anwählen:

document.querySelector('.btn').classList[0];

Tip

The DOM Property Viewer is new in Firefox 48. It is disabled by default. Enable it in the Developer Tools Settings.

mdn

Methoden zur Bearbeitung der Element.classList

MDN Reference
Hinzufügen: document.querySelector('.snippet').classList.add('fire');
Entfernen: document.querySelector('.snippet').classList.remove('fire');
Umschalten: document.querySelector('.snippet').classList.toggle('fire');
Man sollte Elemente nicht mit dem Klassen-Listen Element anwählen das man 'togglen' möchten
Prüfen: document.querySelector('#test').classList.contains('btn');
Attributes bearbeiten:
Prüfen: document.querySelector('#test').hasAttribute('id');
Abfragen: document.querySelector('#test').getAttribute('id');
Ändern: document.querySelector('#test').setAttribute('id', 'newOne');
