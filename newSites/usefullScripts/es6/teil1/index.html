Template Literales

Quasi die Vorstufe zu Template-String. Mit Template Literales, lassen sich wunderbar Strings verbinden:


const first = 'Andi';
const last = 'Arbeit';

const full = `${first} ${last}`;
console.log(full);

Output: Andi Arbeit

Da ich mir nie merken kann, wo auf meiner Tastatur der Backslash versteckt ist, freut es mich toll, dass man in einem Template-String Zeilenumbrüche mirt der Enter-Taste erzeugen kann. Siehe da:


  const umbruch = `${first} ${last}, 
Du bist lustig.`;

document.querySelector('.ziel').innerText = umbruch;

Destruction Objects


  const personalInfo = {
    firstName: 'Andi',
    lastName: 'Arbeit',
    city: 'Offenbach',
    state: 'Ding',
    zipCode: '1234'
  };

Mit dieser Methode kann man sich ordentlich Arbeit sparen. Angebommen wir müssen aus dem oben stehendem Objekt "personalInfo" häufiger auf die Eintträge für Vor und Nachmen zugreifen. Damit man nicht jedes mal auf das Objekt Personal Info stellt man einaml folgendes klar:

const {firstName, lastName} = personalInfo;

Nun bräuchte man nur noch die enstprechenden Variablen aufzurufen:

const {firstName, lastName} = personalInfo;

Die Decostructer-Methode hat aber noch mehr zu bieten. Wir können sie auch nutzen die entsetzlich langen Variablen gegen kürzere zu ersetzen:

const {firstName: fn, lastName: ln} = personalInfo; const {firstName: fn, lastName: ln} = personalInfo; Nun wird die nächste Zeile wirklich kurz und knackig: console.log(`${fn} und auch ${ln}`);
