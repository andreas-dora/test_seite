HTML Input Felder

und wie man sie füttert

HTML stellt eine Reihe verschiedener Input Felder für Zeitangaben zur Verfügung. Möchte man die felder mit Werten aus einer Datenbank oder einem Default-Datum/Zeit füllen darf man sich von ihrem Erscheinungsbild nicht in die Irre führen lassen. Auch wenn es so aussieht als wolle, ein Input Feld vom Typ "date" ein Datum im Format "tt.mm.jjjj" haben. W, wird man schnell sehen, dass es nur als Eingabefeld so funktioniert.

Möchte man das Feld als Ausgabe nutzen, wenn es beispielsweise als default das aktuelle Datum Anzeigen soll, merkt man schnell, dass es Daten in diesem Format nicht annimmt. Zum Glück hilft hier eine entsprechende Fehlermeldung der Console unter Chrome weiter.

inputs.js:8 The specified value "12.11.2021" does not conform to the required format, "yyyy-MM-dd".

Ähnlich verhält es sich mit den Feldern vom Typ "time" und "datetime-local".

type="datetime-local"

inputs.js:9 The specified value "12.11.2021" does not conform to the required format. The format is "yyyy-MM-ddThh:mm" followed by optional ":ss" or ":ss.SSS"

type="date"

type="time"

type="datetime-local"

type="datetime"
