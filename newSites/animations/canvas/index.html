Grundlagen

Der HTML Canvas ist eine großartiges Mittel Webseiten leben einzuhauchen. Diese Leinwand für Animationen und Interaktionen wird von fast allen modernen Browsern unterstützt. Entwicklern bietet das Arbeiten mit dem Canvas eine schöne Gelegenheit ihre Javascript-Skills zu schärfen. Canvas-Animationen lassen sich mit Vanilla Javascript verwirklichen.
